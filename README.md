My final graduation project is a health monitoring system that includes a web application for patients and healthcare providers. 
The system allows patients to track their health data, such as blood pressure, heart rate, and location, while enabling healthcare providers to monitor their patients remotely. 
The project also incorporates features like emergency contact management and real-time location tracking to enhance patient safety and care. 
The project uses SQL for the db, but feel free to use whatever you want to connect it, such as MongoDB, etc. 
Just make sure to alter the config example file to fit your needs.
